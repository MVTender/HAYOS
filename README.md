**HAYOS** is a gorgeous, fast, fully‑offline text editor for Windows. Write in a neon sci‑fi interface, export to Word, PDF or Markdown, and keep everything on your machine — **no account, no ads, no cloud, no telemetry.**

It's a real word processor wrapped in a look you'll actually *want* to write in: a deep‑black page with subtle CRT scan‑lines, glowing text, a glassy toolbar and a living electric line across the title bar.

> 100% free. Tiny download. Yours forever.

---

## ✨ Features

- **Free forever** — freeware, no paywalls, no subscriptions.
- **Lightweight & fast** — opens instantly, runs on the WebView2 engine already built into Windows.
- **Fully offline** — your documents never leave your computer.
- **Beautiful by design** — sci‑fi / CRT interface, typing glow, glassmorphism, custom themes (Night / Minimal).
- **Real document editing** — headings, bold/italic/underline, lists, quotes, **tables** (add/remove rows & columns), links, images.
- **Export anywhere** — Word (`.doc`), RTF, Markdown, TXT, HTML and PDF.
- **Native Save / Open** dialogs, **autosave** and full **session restore**.
- **Spell‑check** — misspelled words glow red, right in the page.
- **Adjustable page margins** — drag the glowing guides on the ruler, or pick a preset.
- **Bilingual interface** — English / Русский, switch anytime.
- **Your own `.hys` format** — a self‑contained HTML file that opens in HAYOS *and* in any browser by double‑click.
- **Zen mode**, command palette, find & replace, live word/character/reading‑time stats.

## ⬇ Download & Install

1. Open the **[latest release](https://github.com/MVTender/HAYOS/releases/latest)** and download `HAYOS-Setup.exe`.
2. Run it — **no administrator rights needed**.
3. Launch **HAYOS** from the Start menu or the desktop shortcut.

**Requirements:** Windows 10 or 11 (64‑bit). Uses the Microsoft **WebView2** runtime, which is preinstalled on Windows 11 (and auto‑installed if missing).

---

## ⌨️ Keyboard shortcuts

| Action | Shortcut |
|---|---|
| Save / Save As | `Ctrl+S` / `Ctrl+Shift+S` |
| Open | `Ctrl+O` |
| Export (Word / RTF / MD / TXT / HTML) | `Ctrl+E` |
| Print → PDF | `Ctrl+P` |
| Find & replace | `Ctrl+F` |
| Command palette | `Ctrl+K` |
| Heading 1 / 2 | `Ctrl+1` / `Ctrl+2` |
| Bold / Italic / Underline | `Ctrl+B` / `Ctrl+I` / `Ctrl+U` |
| Zen mode | `F10` |

---

## 📄 File formats

- **Open:** `.hys`, `.txt`, `.md`, `.html`, `.rtf`
- **Save:** `.hys` (native, portable), plus **Save As** to Word `.doc`, RTF, TXT, Markdown, HTML
- **Export / Print:** PDF (via the system print dialog)

The native **`.hys`** format is just self‑contained HTML — share it and the recipient can open it in any browser, or in HAYOS with full formatting.
